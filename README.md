
# Social Media Platform (Django)

Welcome to the **Social Media Platform** project built using Django!  
This project is a basic social media application where users can register, log in, create posts, explore content, and connect with others.  
It provides a strong foundation that can be extended into a full-featured social networking site.

---

##  Features
- User registration and authentication
- User profiles with profile picture and bio
- Create, edit, and delete posts
- Explore and search posts/users
- Like and follow functionality
- Responsive and clean UI

---

##  Tech Stack
- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite (default, can be extended to MySQL/PostgreSQL)
- **Tools:** VS Code, GitHub

---

## Project Structure
- `manage.py` → Main entry point
- `social_media/settings.py` → Project configuration
- `social_media/urls.py` → URL routing
- `app/models.py` → Database models
- `app/views.py` → Business logic
- `templates/` → HTML templates
